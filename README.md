## 🔥 Featured Project

### 🤖 [VORA – Voice-Oriented Robotic Assistant](https://github.com/varayuts/VORA-robot)

> *An autonomous indoor mobile robot controlled entirely by Thai voice commands*

```
Thai Voice → Whisper STT → Gemma-3 27B (Intent) → Qwen3-VL 8B (Vision) → gTTS → ROS2 Nav2 → Robot
```

| Component | Model / Tech |
|-----------|-------------|
| 🎤 Speech-to-Text | Whisper Thonburian |
| 🧠 LLM (Intent Reasoning) | Gemma-3 27B |
| 👁️ VLM (Scene Understanding) | Qwen3-VL 8B |
| 🔊 Text-to-Speech | gTTS |
| 🤖 Robot Navigation | ROS2 Nav2 · AMCL |
| ⚡ Hardware | Jetson Nano · myAGV |
| 🔌 Backend | FastAPI · WebSocket |
